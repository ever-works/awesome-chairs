## Overview

Dynamic task chair supports natural movement for prolonged sitting.

## Features

- Weight Capacity: 300 lbs
- Back: Flexible elastomer
- Seat: Tilting, depth adjustable
- Adjustments: Height, tilt lock/limit, lumbar height, seat depth
- Armrests: 4D (height, width, angle, depth)
- Materials: Aluminum/polymer base, foam seat
- Ships fully assembled; 12-year warranty

## Use Cases

- Programmers: Versatile for different coding postures
- Streamers: Stable, adjustable for long sessions
