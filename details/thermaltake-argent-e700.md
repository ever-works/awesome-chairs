## Overview

- Genuine top-grain leather
- 4D adjustable armrests (pearl white)
- Dynamic lumbar curve support
- 160° synchronous recline
- Aluminum alloy 5-star base
- High-density foam padding
- Curved ABS back shell with ventilation

Perfect for high-end gaming rigs where style matches performance. Suited for extended sessions in home offices or streaming rooms. Supports up to 330 lbs for versatile user fits.
