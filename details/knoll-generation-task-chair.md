## Overview

The Knoll Generation Task Chair is designed for dynamic, flexible work environments. It features an innovative flex back that pivots and bends together to provide responsive support.

## Features

- Flex back design that moves with the user
- Adjustable seat depth slider
- Height-adjustable arms
- Synchro-tilt mechanism with multiple lock positions
- Pneumatic seat height adjustment
- Breathable mesh back
