## Overview

The Sihoo Doro C300 Pro V2 is a mesh task chair emphasizing dynamic conformability to user posture during movement.

## Features

- Dual independent backrest components: lower supports lumbar and sacrum, upper supports back and shoulders
- Auto-adjusting headrest for neck and head support
- Adjustable backrest height for different torso sizes
- Adjustable seat depth (forward/backward)
- 8D armrests adjustable in multiple angles, foldable sideways
- Recline with rocking and three lock positions: 105°, 120°, 135°
- Optional footrest for $40
