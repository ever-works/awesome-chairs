## Overview

The SitOnIt Novo Task Chair combines a black mesh back with a grey upholstered seat and built-in lumbar support, offering daily ergonomic comfort at an accessible price point.

## Features
- Black mesh back for breathability and airflow
- Grey upholstered seat for cushioned comfort
- Built-in lumbar support for lower back alignment
- Fixed arms for consistent positioning
- Height adjustment via pneumatic cylinder
- Tilt function with tension control
- Ergonomic frame designed to accommodate shorter users

## Best For
- Value-focused users needing reliable ergonomic support
- Shorter individuals who need proportional seating without extensive adjustability
- Budget-conscious home offices and small businesses
- Users preferring fixed armrest positions
