## Overview

The COLAMY 1325F is built for bigger and taller gamers who struggle to find chairs that accommodate their frame. With a wide seat base, 350 lbs weight capacity, and premium materials, it provides real support and lasting comfort regardless of size.

## Specifications

| Feature | Detail |
|---------|--------|
| Weight Capacity | Up to 350 lbs |
| Chair Dimensions | 22" Depth, 26.8" Width, 52.4" Height |
| Materials | Carbon Fiber PU Leather, Memory Foam |
| Frame & Base | Durable Metal Frame, Heavy Duty Metal Base |
| Backrest Reclinability | 90° to 150° |
| Special Features | Adjustable Lumbar Support, Ergonomic Design, Retractable Footrest, Rocking Function |

## Features

- 350 lbs weight capacity built for heavy use
- Extra-wide seat cushion ideal for big and tall users — even allows sitting cross-legged
- Top-tier memory foam seat cushion for premium comfort
- Adjustable lumbar support and ergonomic design
- Retractable footrest for lounger-like leg comfort
- Durable heavy-duty metal base that resists wobble or flex
- Reclines from 90° to 150° with 20° rocking function
