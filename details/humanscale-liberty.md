## Overview

The Humanscale Liberty is a versatile ergonomic office chair combining compact design with Humanscale's signature self-adjusting recline mechanism.

## Features

- Self-adjusting recline — automatically responds to user weight
- Mesh backrest with built-in lumbar
- Flip-up arm pads
- Pneumatic seat height adjustment
- Compact, lightweight frame — great for small offices

Starting at approximately $1,125.
