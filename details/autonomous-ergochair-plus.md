## Overview

ErgoChair Plus uses frameless TPE mesh for adaptive support and airflow during long workdays.

## Features

- Dynamic lumbar via contouring TPE mesh
- Spring-based seat cushioning
- Adjustable arms (7-11 inches)
- Tilt up to 25 degrees
- 300 lbs capacity

## Use Cases

- Coding/programming endurance
- Office meetings and hybrid work

Subtler than gaming chairs, with better professional fit and breathability.

$399 base model.
