## Overview

The Herman Miller Sayl is an office chair designed by Yves Béhar, featuring an innovative 3D Intelligent suspension back that provides ergonomic support.

## Features

- 3D Intelligent back
- Elastomeric suspension
- Adjustable lumbar support
- Fixed arms with 3° forward tilt
- Tilt tension control
- Pneumatic height adjustment

## Specifications

- Weight Capacity: 350 lbs
- Seat Height: 15.5\"–20.25\"
- Warranty: 12-year warranty
