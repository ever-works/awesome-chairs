## Overview

Humanscale World One is an all-mesh ergonomic task chair designed as a lighter, lower-cost option for all-day computer and office work. It uses a mostly self-adjusting recline mechanism with very few manual controls.

## Features
- **All-mesh construction** for seat and back
- **Self-adjusting recline** with no tension knobs
- **Seat height adjustment**
- **Armrest height adjustment**
- **Lightweight design**
