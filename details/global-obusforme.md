## Overview

The Global ObusForme is an ergonomic task chair featuring ObusForme's patented back support technology with comprehensive adjustability including seat angle, depth, and height customization.

## Features

- ObusForme patented ergonomic back support with elastomeric suspension
- Adjustable seat depth for different leg lengths
- Independent back angle adjustment
- Forward/backward seat angle control
- Height adjustable backrest
- Breathable mesh back with supportive padding
- Pneumatic seat height adjustment

## Specifications

- Weight Capacity: Up to 300 lbs
- Seat Depth Adjustable: Multiple positions
- Warranty: Extended commercial warranty
