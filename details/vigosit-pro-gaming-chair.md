## Overview

The Vigosit PRO is a budget gaming chair designed for larger gamers, offering essential ergonomic features without the premium price tag. It delivers reliable support for extended gaming sessions with a focus on comfort and durability.

## Specifications

| Feature | Details |
|---------|--------|
| Weight Capacity | Up to 350 lbs |
| Height Recommendation | 5'6" — 6'4" |
| Seat Width | 21 inches |
| Recline Range | 90° — 155° |
| Material | High-Density Foam with PU Leather |
| Armrests | 2D Adjustable |
| Lumbar Support | Adjustable Cushion |

## Features

- **350-lb Weight Capacity**: Robust frame supports larger users securely
- **21-inch Wide Seat**: Provides ample room without feeling cramped
- **High-Density Foam Padding**: Firm support that holds shape over extended use
- **Adjustable Lumbar Cushion**: Slides into position to ease lower back tension
- **Neck Pillow**: Included head pillow for adjustable neck support
- **PU Leather Upholstery**: Sleek surface that wipes clean easily
- **155° Recline Range**: Allows leaning back for relaxation between sessions
- **2D Adjustable Armrests**: Height and angle adjustments for wrist support
- **Quiet Casters**: Smooth-rolling wheels for easy floor movement
